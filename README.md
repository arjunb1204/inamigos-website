# InAmigos Foundation

Welcome to the official repository for the **InAmigos Foundation** frontend website!

## 🌍 About the Project
InAmigos Foundation is a Section 8 registered non-profit serving India since 2020. This project contains the complete landing page and frontend logic designed to showcase the foundation's mission, their six transformative projects, impact metrics, and volunteer/donation opportunities.

## 🚀 Features
- **Modern UI/UX**: Built with standard HTML, CSS, and vanilla JavaScript.
- **Scroll Animations**: Dynamic scroll reveal animations for a premium user experience.
- **Hero Carousel**: An engaging animated hero section featuring impactful quotes and images.
- **Dynamic Counters**: Animated impact counters that update seamlessly on scroll.
- **Fully Responsive**: Designed to look great on desktop, tablet, and mobile devices.

## 📁 Repository Structure
- `index.html` - The main structure of the site, separated into semantic sections.
- `styles.css` - Custom CSS styling, animations, and typography.
- `script.js` - All interactivity, including the hamburger menu, smooth scrolling, and scroll-triggered animations.
- `inamigos3-Photoroom.png` - Project logo and assets.

## 🛠 Usage
This is a standard static web project. To run it locally:
1. Clone the repository to your local machine.
2. Open `index.html` directly in any modern web browser.
3. No build tools or package managers are required!

## 🤝 Contributing
For updates to the content, feel free to edit the specific section inside `index.html`. For styling updates, utilize the organized classes in `styles.css`.
